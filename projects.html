Clinic Management System Frontend

Designed and developed a full-stack application using .NET, MSSQL and REACT. This project involved creating a responsive, user-friendly design and integrating CRUD for enhanced functionality.

More details

In this project, I developed a robust full-stack application utilizing a .NET C# backend and a React MVC frontend. The application features a seamless integration with a Microsoft SQL Server (MSSQL) database and showcases efficient data handling through a comprehensive API.

Backend Development:

  • Technology: .NET C#
  • Database Integration: Connected the backend to an MSSQL database, ensuring efficient data storage and retrieval.
  • API Development: Created a series of RESTful APIs to handle CRUD operations, enabling communication between the frontend and the database.
  • Testing: Utilized Postman for thorough testing of the APIs, ensuring reliability and correctness in data handling.

Frontend Development:

  • Technology: React MVC
  • State Management: Implemented React Query to manage server state, providing a smooth and responsive user experience.
  • CRUD Operations: Integrated React Query for performing Create, Read, Update, and Delete operations, ensuring efficient data fetching and mutation from the server.
  • UI/UX: Developed a user-friendly interface that aligns with modern design principles and enhances the overall user experience.

Key Features:

  • Robust Architecture: Leveraged the power of .NET and React to build a scalable and maintainable application architecture.
  • Efficient Data Handling: Employed advanced querying techniques with React Query to optimize data fetching and synchronization with the backend.
  • Comprehensive Testing: Ensured API reliability and performance through extensive testing with Postman.

Tools & Technologies:

  • Backend: .NET C#, MSSQL, Entity Framework
  • Frontend: React, React Query, HTML5, CSS3, JavaScript (ES6+)
  • Testing: Postman
  • Version Control: Git
